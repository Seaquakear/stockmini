### stockmini
**一款基于vue2与element-ui开发的股票简易数据展示工具。**
1、运行前准备
下载好下项目后，先执行npm install命令，来下载项目需要用到的包，可能有130MB左右的包。
2、运行项目
输入命令：npm run serve来运行项目，项目运行起来后效果如图所示。
